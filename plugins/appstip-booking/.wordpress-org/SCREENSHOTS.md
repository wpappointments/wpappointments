# Appstip Booking — wp.org listing screenshots

The wp.org listing requires up to 5 PNG screenshots, indexed `screenshot-1.png`
through `screenshot-5.png`, with captions matching the `== Screenshots ==`
section in `readme.txt`.

## Files

Drop final PNGs in this directory under these exact names:

| File | Caption (matches readme.txt) |
|---|---|
| `screenshot-1.png` | Appointments admin list — manage your appointment records from a familiar WordPress screen. |
| `screenshot-2.png` | Editing a single appointment — the standard block editor with title and content. |
| `screenshot-3.png` | Settings page — set the default appointment duration. |

## Generation

Auto-generated by Playwright spec at `tests/e2e/appstip-booking/screenshots.spec.ts`.

Run on CI (after this branch lands) or locally if Playwright resolves cleanly:

```bash
WPORG_SCREENSHOTS=1 pnpm -C plugins/wpappointments test:e2e -- appstip-booking
```

Output lands in `tests/e2e/results/appstip-booking/`. Copy the three PNGs
into `.wordpress-org/` for SVN sync.

## Dimensions

wp.org accepts any reasonable size; 1280×800 is the common choice.
Keep them consistent — mixed sizes look uneven in the listing carousel.
