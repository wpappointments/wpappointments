Appointments Booking provides actions and filters throughout the plugin for extensibility.

:::note
The hooks listed in [Actions](/developers/hooks/actions/) and [Filters](/developers/hooks/filters/) are auto-generated from the source code. Run `pnpm docs:generate` to update them.
:::

## Actions

Actions fire at specific points in the plugin lifecycle. Use `add_action()` to hook into them.

See the full [Actions reference](/developers/hooks/actions/).

## Filters

Filters let you modify data before it's used. Use `add_filter()` to hook into them.

See the full [Filters reference](/developers/hooks/filters/).
