Appointments are the core entity in WP Appointments. Each appointment links a customer to a bookable service at a specific time.

## From the Admin

1. Go to **WP Appointments → Calendar**
2. Click a time slot or use the **New Appointment** button
3. Select a service, customer, date, and time
4. Click **Save**

## From the Booking Flow

Customers can book appointments through the **Booking Flow** Gutenberg block on your site. The flow guides them through service selection, date/time picking, and confirmation.

## Appointment Statuses

| Status | Description |
|--------|-------------|
| **Confirmed** | Appointment is confirmed and scheduled |
| **Pending** | Awaiting confirmation |
| **Cancelled** | Appointment was cancelled |
| **No-show** | Customer did not attend |
