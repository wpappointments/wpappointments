The **Booking Flow** block lets customers book appointments directly from your site.

## Adding the Block

1. Edit a page or post in the block editor
2. Add a new block and search for **Booking Flow**
3. Configure the block settings in the sidebar
4. Publish the page

## Layout Options

| Layout | Description |
|--------|-------------|
| **Single Step** | All fields on one page |
| **Multi Step** | Guided wizard with step-by-step flow |

## How It Works

The booking flow guides customers through:
1. **Service selection** — choose what to book
2. **Date & time** — pick an available slot
3. **Customer info** — enter name, email, phone
4. **Confirmation** — review and submit
