WP Appointments publishes shared npm packages that addon developers can use to build admin UIs and integrate with the data layer.

## Available Packages

| Package | Version | Description |
|---------|---------|-------------|
| [`@wpappointments/components`](/packages/components/) | ![npm](https://img.shields.io/npm/v/@wpappointments/components) | React UI components for admin interfaces |
| [`@wpappointments/data`](/packages/data/) | ![npm](https://img.shields.io/npm/v/@wpappointments/data) | Hooks, API client, type registry, utilities |

## Installation

```bash
npm install @wpappointments/components @wpappointments/data
```

## Versioning

Packages are versioned independently from the main plugin using [Changesets](https://github.com/changesets/changesets). Check the package CHANGELOG for breaking changes.
