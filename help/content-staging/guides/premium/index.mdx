Premium addons extend Appointments Booking with advanced capabilities. All premium addons are included in a single subscription.

## Available Addons

| Addon | Description |
|-------|-------------|
| **Employees** | Multi-staff scheduling with individual calendars |
| **Locations** | Multi-location support with per-location availability |
| **Payments** | WooCommerce integration for paid bookings |
| **Custom Fields** | Add custom fields to booking forms |

## Getting Premium

Visit [wpappointments.com](https://wpappointments.com) for pricing and details.
