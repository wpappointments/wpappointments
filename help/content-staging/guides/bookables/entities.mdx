Bookable entities are the things your customers book. The core plugin ships with a **Service** type, but the system is fully extensible — addons can register custom bookable types like rooms, courts, or equipment.

## Creating a Bookable Entity

1. Go to **Appointments Booking → Services** (or the relevant bookable type page)
2. Click **Add New**
3. Fill in the name, duration, and description
4. Configure availability if different from global schedule
5. Save

## Fields

| Field | Description |
|-------|-------------|
| **Name** | Display name shown to customers |
| **Duration** | Default appointment length |
| **Description** | Shown in the booking flow |
| **Status** | Active or inactive |
