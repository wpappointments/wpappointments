Variants let you offer different options for the same bookable entity. For example, a "Haircut" service might have variants for "Quick Trim" (15 min) and "Full Style" (45 min).

## Creating Variants

1. Open a bookable entity
2. Go to the **Variants** tab
3. Click **Add Variant**
4. Set the variant name, duration override, and any custom fields
5. Save

## Auto-Generation

For bookable types that support it, you can auto-generate variants from a matrix of attributes (e.g., surface type × time slot for a court booking system).
