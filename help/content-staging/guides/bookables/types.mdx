Appointments Booking uses a **Bookable Type Registry** — a system that lets addons define entirely new categories of bookable entities beyond the built-in Service type.

## Built-in Types

| Type | Description |
|------|-------------|
| **Service** | The default bookable type (haircuts, consultations, etc.) |

## Custom Types (via Addons)

Addons can register custom types like:
- **Court** — for sports facility booking
- **Room** — for meeting room reservations
- **Equipment** — for rental systems

Each custom type gets its own admin page, form fields, and REST API endpoints automatically.

See the [Addon Development](/developers/addons/) guide for how to register a custom bookable type.
