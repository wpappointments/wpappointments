Appointments Booking is a WordPress plugin for managing appointment scheduling. It provides a calendar, customer management, booking flow, and email notifications out of the box.

## Quick Start

1. Install the plugin from the WordPress plugin directory or upload the ZIP
2. Activate **Appointments Booking** in Plugins
3. Run the onboarding wizard at **Appointments Booking → Dashboard**
4. Add the **Booking Flow** block to any page or post

## What's Included

| Feature | Description |
|---------|-------------|
| **Appointments** | Create, edit, cancel appointments with status tracking |
| **Calendar** | Visual calendar view with day/week/month layouts |
| **Customers** | Customer database with appointment history |
| **Bookables** | Define what can be booked (services, entities, variants) |
| **Availability** | Layered schedule system with opening hours and blocks |
| **Notifications** | Email notifications with template variables |
| **Booking Flow** | Gutenberg block for customer-facing booking |
| **REST API** | Full API for headless and custom integrations |
