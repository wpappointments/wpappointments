## Editing an Appointment

Click any appointment in the calendar or appointments list to open the detail view. From there you can:

- Change the date and time
- Reassign to a different customer
- Update the status
- Add notes

## Cancelling

Set the status to **Cancelled**. The customer receives a cancellation notification if email notifications are enabled.

## Calendar Views

The calendar supports day, week, and month views. Switch between them using the view controls in the top-right corner.
