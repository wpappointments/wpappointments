description: How to install and activate WP Appointments.
---

## Requirements

- WordPress 6.4 or later
- PHP 8.2 or later

## Install from WordPress

1. Go to **Plugins → Add New**
2. Search for **WP Appointments**
3. Click **Install Now**, then **Activate**

## Install from ZIP

1. Download the latest release from [GitHub Releases](https://github.com/wpappointments/wpappointments/releases)
2. Go to **Plugins → Add New → Upload Plugin**
3. Upload the ZIP file and click **Install Now**
4. Activate the plugin

## First-Time Setup

After activation, go to **WP Appointments → Dashboard**. The onboarding wizard walks you through:

1. **General settings** — business name, timezone
2. **Schedule** — opening hours per day of week
3. **Services** — define your first bookable service
4. **Appointments** — configure default duration and buffer times

## Embedding the Booking Flow

Add the **Booking Flow** Gutenberg block to any page or post. It supports single-step and multi-step layouts.
